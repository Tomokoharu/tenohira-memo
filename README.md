# てのひらメモ

訪問前10秒で患者さんを思い出すための、iPhone向け静的プロトタイプです。

## GitHub Pagesで公開する手順

1. このフォルダの中身をGitHubリポジトリに入れます。
2. GitHubでリポジトリを開き、`Settings` → `Pages` を開きます。
3. `Build and deployment` の `Source` を `Deploy from a branch` にします。
4. `Branch` は `main`、フォルダは `/root` を選んで `Save` します。
5. 数分後に表示されるURLを開くと使えます。

## 大事な注意

- 入力した患者メモは、サーバーではなく開いた端末のブラウザ内に保存されます。
- 同じURLでも、別のiPhoneや別のブラウザには自動共有されません。
- ブラウザの履歴やサイトデータを消すと、保存したメモも消えることがあります。
- 実名、詳しい住所、保険情報、詳細病名は入れすぎない前提のメモアプリです。

## ホーム画面に追加

iPhoneのSafariで公開URLを開き、共有ボタンから「ホーム画面に追加」を選びます。
